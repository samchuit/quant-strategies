# 股票量化策略

## 概述
股票市场量化策略，涵盖A股、美股、港股等主要市场。

## 策略分类

### 趋势跟踪策略
1. **双均线策略 (Dual Moving Average)**
   - 核心逻辑: 短期均线上穿长期均线时买入，下穿时卖出
   - 技术指标: SMA, EMA
   - 复杂度: 基础

2. **唐奇安通道策略 (Donchian Channel)**
   - 核心逻辑: 突破N日最高价买入，跌破N日最低价卖出
   - 技术指标: 最高价、最低价
   - 复杂度: 基础

3. **布林带突破策略 (Bollinger Bands Breakout)**
   - 核心逻辑: 价格突破布林带上轨买入，跌破下轨卖出
   - 技术指标: 布林带 (Bollinger Bands)
   - 复杂度: 基础

### 均值回归策略
1. **RSI超买超卖策略**
   - 核心逻辑: RSI低于30超卖买入，高于70超买卖出
   - 技术指标: RSI (Relative Strength Index)
   - 复杂度: 基础

2. **配对交易策略 (Pairs Trading)**
   - 核心逻辑: 寻找相关性高的股票对，做多低估股票，做空高估股票
   - 技术指标: 协整关系、价差Z-score
   - 复杂度: 中级

### 多因子模型
1. **Fama-French三因子模型**
   - 核心逻辑: 市场因子、规模因子、价值因子
   - 技术指标: 市值、账面市值比
   - 复杂度: 中级

2. **Carhart四因子模型**
   - 核心逻辑: 在三因子基础上加入动量因子
   - 技术指标: 动量效应
   - 复杂度: 中级

### 机器学习策略
1. **LSTM股价预测**
   - 核心逻辑: 使用LSTM神经网络预测股价走势
   - 技术指标: 时序特征、技术指标
   - 复杂度: 高级

2. **XGBoost多因子选股**
   - 核心逻辑: 使用梯度提升树模型进行选股
   - 技术指标: 基本面因子、技术因子
   - 复杂度: 高级

## 数据需求
- 日级/分钟级行情数据
- 财务数据 (PE, PB, ROE等)
- 市值、成交量数据
- 融资融券数据 (A股)

## 常用工具库
- `akshare` - 中国股票数据
- `yfinance` - 美股数据
- `backtrader`, `zipline` - 回测框架
- `ta-lib` - 技术指标计算
- `scikit-learn`, `tensorflow` - 机器学习

## 风险提示
- 股票市场存在系统性风险
- 策略存在过拟合风险
- 需考虑交易成本、滑点影响
- 注意市场风格切换
