# 期货量化策略

## 概述
期货市场量化策略，涵盖商品期货、金融期货、股指期货等。

## 策略分类

### CTA策略 (商品交易顾问)
1. **海龟交易法则 (Turtle Trading)**
   - 核心逻辑: 突破N日高低点入场，ATR止损
   - 技术指标: 突破点、ATR (平均真实波幅)
   - 复杂度: 基础
   - 适用品种: 商品期货、股指期货

2. **Dual Thrust策略**
   - 核心逻辑: 基于开盘价和收盘价计算上下轨，突破上轨做多，突破下轨做空
   - 技术指标: 最高价、最低价、开盘价、收盘价
   - 复杂度: 基础
   - 适用品种: 股指期货、商品期货

### 统计套利策略
1. **期现套利 (Cash-Futures Arbitrage)**
   - 核心逻辑: 利用期货与现货价格差异套利
   - 技术指标: 基差、持有成本
   - 复杂度: 中级
   - 适用品种: 股指期货、国债期货

2. **跨期套利 (Calendar Spread)**
   - 核心逻辑: 同一品种不同到期月份合约的价差交易
   - 技术指标: 价差、季节性因素
   - 复杂度: 中级
   - 适用品种: 商品期货、股指期货

3. **跨品种套利 (Inter-commodity Spread)**
   - 核心逻辑: 相关品种间的价差交易 (如金/银、豆油/豆粕)
   - 技术指标: 相关性、协整关系
   - 复杂度: 中级
   - 适用品种: 商品期货

### 高频策略
1. **做市商策略 (Market Making)**
   - 核心逻辑: 同时挂买单和卖单，赚取买卖价差
   - 技术指标: 订单簿数据、价差
   - 复杂度: 高级
   - 适用品种: 流动性好的期货品种

2. **闪电交易 (Flash Trading)**
   - 核心逻辑: 利用极短时间窗口的价格异常
   - 技术指标: 毫秒级行情数据
   - 复杂度: 高级
   - 适用品种: 股指期货、国债期货

### 基本面量化策略
1. **库存周期策略**
   - 核心逻辑: 基于商品库存数据预测价格走势
   - 技术指标: 库存数据、供需平衡表
   - 复杂度: 中级
   - 适用品种: 商品期货

2. **季节性策略**
   - 核心逻辑: 利用商品价格的季节性规律
   - 技术指标: 历史季节性模式
   - 复杂度: 基础
   - 适用品种: 农产品期货

## 期货特性考虑
- 杠杆效应 (保证金交易)
- 交割机制
- 展期成本 (Rollover Cost)
- 合约流动性差异
- 交易时间差异 (夜盘/日盘)

## 数据需求
- Tick级/分钟级行情数据
- 持仓量、成交量数据
- 基本面数据 (库存、产量等)
- 宏观经济数据

## 常用工具库
- `vnpy` - 国内期货交易框架
- `backtrader` - 回测框架
   - 支持期货合约展期
   - 支持保证金计算
- `tqsdk` - 天勤量化
- `rqdatac` - RiceQuant数据

## 风险提示
- 高杠杆带来的高风险
- 流动性风险 (特别是远月合约)
- 交割风险
- 政策风险 (交易规则变化)
